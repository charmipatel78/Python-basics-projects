# Visualizer

## Description

This project is a Python-based data analysis and visualization tool that uses Pandas and Matplotlib to explore and visualize data from CSV files.

## Features

* Load and display dataset
* Show first and last rows of data
* Display dataset information and summary
* Perform basic data analysis (describe, shape, columns)
* Filter and sort data
* Data visualization:

  * Line chart
  * Bar chart
  * Scatter plot
  * Pie chart
* Save visualizations as image files

## Concepts Used

* Pandas (data handling)
* Matplotlib (data visualization)
* File handling (CSV)
* Data filtering and sorting
* Loops and conditional statements
* User input handling

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Install required libraries:
   pip install pandas matplotlib
3. Run the program
4. Choose options from the menu:

   * 1 → Load dataset
   * 2 → Explore data
   * 3 → Perform operations
   * 4 → Visualize data
   * 5 → Save visualization
   * 6 → Exit
