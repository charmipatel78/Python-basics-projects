# Logic Box

## Description

This project is a logic-based Python program that provides multiple functionalities using a menu-driven approach. It helps improve problem-solving and logical thinking skills.

## Features

* Menu-driven program
* Pattern generation using loops
* Number analysis (even/odd check)
* Sum calculation of numbers in a range
* Input validation using try-except

## Concepts Used

* Functions
* Loops (for loop)
* Conditional statements (if-else)
* User input
* Exception handling
* Range and iteration

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Run the program
3. Select an option from the menu:

   * 1 → Generate pattern
   * 2 → Analyze numbers
   * 3 → Exit
