# Analyzer

## Description

This project is a Python-based data analyzer that uses NumPy to perform various operations on numerical data. It provides multiple functionalities through a menu-driven interface.

## Features

* Create and store NumPy arrays
* Calculate sum and mean of data
* Filter even and odd numbers
* Search for specific values
* Perform aggregate operations (min, max, standard deviation, variance)
* Menu-driven program

## Concepts Used

* NumPy library
* Arrays and numerical operations
* Functions
* Conditional statements
* Loops
* User input handling

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Install NumPy (if not installed):
   pip install numpy
3. Run the program
4. Choose options from the menu:

   * 1 → Create Array
   * 2 → Perform Summary
   * 3 → Filter Data
   * 4 → Search Data
   * 5 → Aggregate Statistics
   * 6 → Exit
