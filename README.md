# Python-basics-projects

# Python Basics Projects

This repository contains a collection of beginner to intermediate level Python projects. These projects are created to build strong fundamentals in Python programming.

## 📂 Project Categories

### 1. Fundamentals Booster

Basic programs to strengthen core Python concepts like loops, conditions, and functions.

### 2. Logic Box

Programs focused on logic building and problem-solving.

### 3. File Operator

Projects that perform file handling operations like reading and writing files.

### 4. Collection Manipulator

Programs that work with Python collections such as lists, tuples, sets, and dictionaries.

### 5. Functional Treat

Projects demonstrating the use of functions and modular programming.

### 6. Analyzer

Basic data analysis programs using Python.

### 7. Visualizer

Simple data visualization using libraries like matplotlib.

### 8. Modular & Packager

Projects demonstrating modular coding and packaging in Python.

### 9. OOP Wrapper

Programs based on Object-Oriented Programming concepts like classes and objects.

## 🛠 Technologies Used

* Python 3

## 🎯 Purpose

The goal of this repository is to improve Python programming skills by practicing different types of problems and real-world mini projects.

## 🚀 How to Use

1. Open any folder
2. Run the `.py` files using PyCharm or any Python IDE
3. Explore different concepts
