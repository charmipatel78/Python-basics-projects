# Collection Manipulator

## Description

This project is a Python-based student data management system that uses collections like dictionaries and lists to store and manage student records.

## Features

* Add new student records
* Display all students
* Update student details (age and subjects)
* Delete student records
* Display all subjects offered
* Menu-driven interface

## Concepts Used

* Dictionaries
* Lists
* Functions
* Loops
* Conditional statements
* User input handling
* Date formatting (datetime)

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Run the program
3. Choose an option from the menu:

   * 1 → Add student
   * 2 → Display all students
   * 3 → Update student
   * 4 → Delete student
   * 5 → Display subjects offered
   * 6 → Exit
