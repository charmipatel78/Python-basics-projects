# Functional Treat

## Description

This project is a Python-based data analyzer and transformer that performs various operations on a list of numbers using functions and built-in features.

## Features

* Accepts user input as a list of numbers
* Displays summary (min, max, sum, average)
* Calculates factorial of a number
* Filters data using threshold (lambda function)
* Sorts data in ascending and descending order
* Menu-driven interface

## Concepts Used

* Functions
* Lists
* Lambda functions
* Built-in functions (min, max, sum)
* Loops
* Conditional statements
* User input handling

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Run the program
3. Choose options from the menu:

   * 1 → Input data
   * 2 → Display summary
   * 3 → Calculate factorial
   * 4 → Filter data
   * 5 → Sort data
   * 6 → Exit

## Example Output

Input data: 1 2 3 4 5

Summary:
Min: 1
Max: 5
Sum: 15
Average: 3

Sorted (Ascending): 1 2 3 4 5

## Purpose

To practice functional programming concepts and perform data operations using Python.
