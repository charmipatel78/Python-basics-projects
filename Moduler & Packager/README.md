# Moduler & Packager

## Description

This project is a modular Python application that combines multiple functionalities into a single menu-driven system. It demonstrates the use of modules and organized code structure.

## Features

* General operations (date, time, calculations)
* Math operations (factorial, square root, etc.)
* Random data generation (numbers, passwords)
* File handling (create, read, append, delete files)
* System utilities (menu-driven navigation)

## Concepts Used

* Modular programming
* Functions
* Python standard libraries (math, random, datetime, os)
* File handling
* Loops and conditional statements
* User input handling

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Run the main program
3. Choose options from the menu:

   * 1 → General Operations
   * 2 → Math Operations
   * 3 → Random Generator
   * 4 → File Operations
   * 5 → Exit
