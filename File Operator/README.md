# File Operator

## Description

This project is a Python-based journal manager that allows users to store, view, search, and delete journal entries using file handling.

## Features

* Add journal entries with timestamp
* View all saved entries
* Search entries using keywords
* Delete all entries
* Menu-driven interface
* Exception handling for file operations

## Concepts Used

* File handling (read, write, append)
* Classes and Objects (OOP)
* Functions
* Exception handling (try-except)
* Date and time (datetime module)
* String operations

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Run the program
3. Choose options from the menu:

   * 1 → Add new entry
   * 2 → View all entries
   * 3 → Search entries
   * 4 → Delete all entries
   * 5 → Exit
