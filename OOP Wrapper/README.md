# OOP Wrapper

## Description

This project is a Python-based employee management system built using Object-Oriented Programming (OOP) concepts. It demonstrates the use of classes, inheritance, and method overriding.

## Features

* Create a Person object
* Create an Employee (inherits from Person)
* Create a Manager (inherits from Employee)
* Display details of objects
* Compare employee salaries
* Menu-driven interface

## Concepts Used

* Classes and Objects
* Inheritance
* Method Overriding
* Encapsulation
* Functions
* Conditional statements
* User input handling

## Project Structure

* person.py → Base class (Person)
* employee.py → Derived class (Employee)
* manager.py → Derived class (Manager)
* show_details.py → Displays object details
* OOP Wrapper.py → Main program

## How to Run

1. Open the project in PyCharm or any Python IDE
2. Run the main file (OOP Wrapper.py)
3. Choose options from the menu:

   * 1 → Create Person
   * 2 → Create Employee
   * 3 → Create Manager
   * 4 → Show Details
   * 5 → Compare Salaries
   * 6 → Exit
