# Fundamentals Booster

## Description

This project is a basic Python program designed to strengthen fundamental programming concepts. It takes user input, performs type conversions, and displays useful information.

## Features

* Takes user input (name, age, height, favourite number)
* Displays formatted output
* Demonstrates type conversion (int, float, string)
* Shows data types and memory addresses
* Calculates approximate birth year

## Concepts Used

* Variables
* Input/Output
* Data Types
* Type Casting
* f-strings
* Basic Arithmetic

## How to Run

1. Open the file in PyCharm or any Python IDE
2. Run the program
3. Enter the required inputs when prompted

Enter your name: Charmi
Enter your age: 20
Enter your height cm: 160

## Purpose

To build a strong foundation in Python by practicing core concepts.
